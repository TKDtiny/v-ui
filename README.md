# vue-multipage
# 2017年9月10日提交，所用皆为最新版构建
> * vuejs为2.4.2
> * webpack为2.3.2
> * stylus等
> * ...

# 使用方式
> * 打开命令终端并进入vue-multipage
> * 编译 npm run build  会生成dist目录
> * 运行 npm run dev 浏览器输入localhost:8080/module/shop.html访问
